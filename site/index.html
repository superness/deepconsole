Quickstart

DeepConsole is the UI; it runs against a sibling backend called abuddi (a DeepSeek-backed agent server). You clone both side by side.

Prerequisites

  • Node.js 18+ (tested with v24)
  • Python 3.10+
  • A DeepSeek API key — from platform.deepseek.com

Setup

  1. Clone both repos as siblings (same parent folder): 
    git clone https://github.com/superness/deepconsole.git
git clone https://github.com/superness/abuddi.git
  2. Install the backend's Python dependencies: 
    cd abuddi
pip install -r requirements.txt
  3. Add your DeepSeek key — create abuddi/.env: 
    DEEPSEEK_API_KEY=sk-your-key-here
  4. Install and launch the app: 
    cd ../deepconsole
npm install
npm start
  5. That's it. npm start launches Electron, which spawns the abuddi backend on port 8000 and waits for it to report healthy. When the app shows ready, start chatting.

Folder layout

After cloning, your parent folder should contain deepconsole/ and abuddi/ next to each other. The app looks for the backend at ../abuddi relative to itself.

Hitting a snag? See Troubleshooting. Run npm run dev instead of npm start to open with DevTools.
